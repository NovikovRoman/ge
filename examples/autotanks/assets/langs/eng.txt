##tutorial.keymap.keyboard
CONTROL SCHEME: KEYBOARD

IN MENU:

    [W], [S] - selects the focused menu button
    [A], [D] - loop through the focused button options
    [Enter] - activate button / toggle option
    [Escape] - go to the previous screen

IN BATTLE:

    [W], [A], [S], [D] - move the sector selector
    [Space] - select units / send units
    [Enter] - open sector menu
    [E] - build a defensive turret
    [Q] - cancel selection / action
    [Escape] - back to the menu

IN BATTLE SECTOR MENU:

    [W], [S] - select turret/hull entry
    [A], [S] - loop through the selected entry options
    [Space] - order selected design
    [Enter], [Q] - close the menu

##tutorial.keymap.gamepad
CONTROL SCHEME: GAMEPAD (X-BOX LAYOUT)

IN MENU:

    [Up], [Down] - selects the focused menu button
    [Left], [Right] - loop through the focused button options
    (A) - activate button / toggle option
    [Select] - go to the previous screen

IN BATTLE:

    [Up], [Down], [Left], [Right] - move the sector selector
    (A) - select units / send units
    (X) - open sector menu
    (Y) - build a defensive turret
    (B) - cancel selection / action
    [Select] - back to the menu

IN BATTLE SECTOR MENU:

    [Up], [Down] - select turret/hull entry
    [Left], [Right] - loop through the selected entry options
    (A) - order selected design
    (X), (B) - close the menu

##tutorial.hints
HOW TO PLAY: A QUICK GUIDE

    * Send the starting builder to some nearby sector
    * Build more builders to expand, sectors provide resources
    * Order some battle tanks for attack and defense purposes
    * When your base is under attack, build a defensive turret

You can't control your units directly,
but you can choose a sector to send them to.

Every unit consist of two parts: a hull and a turret.
Depending on what resources and situation you have,
different combinations can be considered optimal.
Open "UNIT STATS" page to see the relevant values.

Good luck and have fun!
